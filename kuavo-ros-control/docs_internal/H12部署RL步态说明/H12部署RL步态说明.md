## RL + H12 + VR

## 部署

- 脚本：已集成到原本的 ocs2 部署脚本 `deploy_autostart.sh`

- 运行前：
  
  - RL 步态需要手动修改 bashrc 中的 ROBOT_VERSION=46
  
  - 确认 Kuavo-RL 仓库的绝对路径，修改脚本中的变量：`KUAVO_RL_WS_PATH`
  
  - 使用 VR 需要修改 `kinematic_mpc_vr_for_rl.launch` 中的 `ip_address` 参数

- 运行开始，输入 1 或 2 选择部署 rl 还是 ocs2

- 切换步态模式重新运行脚本需选择对应的模式

## H12 使用方案

- 标定：E左 + F右 + D

- 下蹲：E左 + F右 + C

- 起立：E左 + F右 + C

- 进入 RL 模式：E中 + F中 + C

- 走路：E中 + F中 + A

- 踏步：E中 + F中 + B

- 站立：E中 + F中 + D

- 进入 VR (站立状态)：长按 A

- 退出 VR (站立状态)：长按 B

- 执行动作 (站立状态)：与原来一致

- 退出程序 (任意状态)：同时长按 C + D

## 状态切换描述

```
                                | <--> 踏步
标定 -> 下蹲 -> 起立 -> RL模式 -> 走路 
                                | <--> 站立 <--> VR遥操
                                        | <--> 执行动作
```

- 踏步模式只能和走路来回切换，通过 `E中 + F中 + B`，所以只能先进入走路再变为踏步
- 踏步转换到走路只是速度为零的走路，不是上锁的站立，若要站立，需要进入走路再变为站立
- 踏步模式下可以增加摇杆变成速度大的走路，松摇杆则继续踏步

## VR 使用方案

- 推摇杆即走，释放即停

- 左手柄控制前进后退和左右平移，右手柄控制转向

- 机器人行走结束，按下 A，可进行 VR 遥操作

- 遥操作结束之前，建议将机器人手臂手动归位，结束之后，推摇杆即可走路

- 站立状态下
  
  - 同时长按左右上扳机解锁手臂，进行 VR 遥操作
  
  - 同时长按左右下扳机固定手臂，此时挥动手臂机器人不动

- 触摸 X 或 Y 分别控制左、右手大拇指旋转

- 左、右上扳机按下和释放分别控制左、右手五指握紧和展开

- 按 Y 可以锁定手指的位置，再按一次解锁

- 同时按下 X + Y 结束机器人程序
