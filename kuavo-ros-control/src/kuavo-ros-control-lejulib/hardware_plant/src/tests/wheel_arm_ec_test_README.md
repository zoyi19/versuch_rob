# 轮臂EC测试工具文档

## 概述

`wheel_arm_ec_test` 是一个用于测试轮臂硬件控制的交互式测试工具，基于 EtherCAT (EC) 通信协议。该工具提供了完整的轮臂系统测试功能，包括关节状态监控、单个关节控制和硬件状态检查。

## 主要功能

### 1. 硬件初始化
- 自动检测 `ROBOT_VERSION` 环境变量
- 初始化轮臂硬件系统
- 配置 kuavo_assets 路径

### 2. 关节监控
- **轮臂腿部关节 (4个)**
- **手臂关节 (14个)**
- **头部关节 (2个)**
- 支持角度和弧度两种单位显示

### 3. 关节控制
- 单个关节精确控制
- 实时关节角度反馈
- 平滑关节运动控制

## 使用方法

### 编译
```bash
cd <kuavo-ros-control>
catkin build hardware_node
```

### 运行
```bash
./devel/lib/hardware_node/hardware_plant_wheel_arm_test 
```


## 交互式操作

程序启动后会显示交互式菜单：

```
[WheelArmECTest] 测试菜单:
按下 'i' 测试单个关节控制
按下 'c' 检查EC状态和当前关节角度
按下 'p' 打印当前关节角度
按下 'q' 退出
```

### 操作说明

#### 1. 单个关节控制 ('i')
- 输入关节ID (1-20)
  - 1-4: 轮臂腿部关节
  - 5-18: 手臂关节
  - 19-20: 头部关节
- 输入目标角度（度）
- 系统将平滑移动关节到目标位置

#### 2. EC状态检查 ('c')
- 显示所有关节的电机状态
- 打印完整的关节角度信息
- 检查 EtherCAT 通信状态

#### 3. 打印关节角度 ('p')
- 实时显示所有关节的当前角度
- 同时显示度数和弧度值
- 按关节类型分组显示

#### 4. 退出程序 ('q')
- 安全退出测试程序
- 自动清理硬件资源
