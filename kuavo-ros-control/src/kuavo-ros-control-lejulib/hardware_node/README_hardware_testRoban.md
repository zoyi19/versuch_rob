
# 硬件波形测试程序使用说明-Roban

---

## 概述

`hardware_testRoban` 是一个用于机器人硬件测试的 ROS 程序，主要用于对机器人关节电机进行波形测试。程序支持方波、三角波和正弦波三种波形类型，可以实时调整波形参数（幅度、频率、偏移量），并通过力矩控制模式驱动指定电机。

**程序启动后会自动将机器人移动到校准姿态（全零位置），然后可以通过键盘命令进入测试模式，对指定电机施加波形力矩信号。**

---

## 功能特性

* **✅ ****三种波形类型**：方波、三角波、正弦波
* **✅ ****实时参数调整**：幅度、频率、偏移量可实时调整
* **✅ ****电机选择**：支持选择左侧或右侧的任意电机（1-6号）
* **✅ ****力矩控制**：使用力矩控制模式（CST）驱动电机
* **✅ ****自动数据记录**：通过 rosbag 自动记录所有测试数据
* **✅ ****安全保护**：测试退出时自动返回校准姿态
* **✅ ****实时反馈**：每0.5秒打印一次测试状态

## 启动方法

### 基本启动

```
 roslaunch hardware_node hardware_testRoban.launch
```

### 带参数启动

```
 # 指定机器人版本
 roslaunch hardware_node hardware_testRoban.launch robot_version:=14
 
 # 启用 RViz 可视化
 roslaunch hardware_node hardware_testRoban.launch rviz:=true
 
 # 启用多图可视化
 roslaunch hardware_node hardware_testRoban.launch multiplot:=true
 
 # 校准模式
 roslaunch hardware_node hardware_testRoban.launch cali:=true
 
 # 组合参数
 roslaunch hardware_node hardware_testRoban.launch robot_version:=14 rviz:=true
```

### Launch 文件参数说明

| **参数**    | **默认值** | **说明**                                             |
| ----------------- | ---------------- | ---------------------------------------------------------- |
| `robot_version` | `14`           | **机器人版本号（从环境变量** `ROBOT_VERSION`读取） |
| `rviz`          | `false`        | **是否启动 RViz 可视化**                             |
| `multiplot`     | `false`        | **是否启动多图可视化工具**                           |
| `cali`          | `false`        | **是否启用校准模式**                                 |
| `real`          | `true`         | **是否真实硬件模式**                                 |
| `log`           | `true`         | **是否记录 LCM 日志**                                |

## 键盘操作说明

### 基本操作

| **按键** | **功能**         | **说明**                   |
| -------------- | ---------------------- | -------------------------------- |
| `t`          | **进入测试模式** | **开始发送波形控制命令**   |
| `q`          | **退出测试模式** | **停止测试，返回校准姿态** |
| `x`          | **退出程序**     | **完全退出程序**           |
| `h`          | **显示帮助**     | **显示所有可用命令**       |

### 波形类型切换（测试模式下）

| **按键** | **波形类型** | **说明**                     |
| -------------- | ------------------ | ---------------------------------- |
| `1`          | **方波**     | **在测试模式下切换为方波**   |
| `2`          | **三角波**   | **在测试模式下切换为三角波** |
| `3`          | **正弦波**   | **在测试模式下切换为正弦波** |

**注意**：在测试模式下，`1`-`3` 用于切换波形类型。要选择电机，请使用 `m` 进入电机选择模式，或使用 `4`-`9`。

### 波形参数调整

| **按键**   | **功能**     | **步长**    | **范围**                  |
| ---------------- | ------------------ | ----------------- | ------------------------------- |
| `w`            | **增加幅度** | **+0.1 Nm** | **≥ 0**                  |
| `s`            | **减少幅度** | **-0.1 Nm** | **≥ 0**                  |
| `e`            | **增加频率** | **+0.1 Hz** | **0.1 - 10.0 Hz**         |
| `d`            | **减少频率** | **-0.1 Hz** | **0.1 - 10.0 Hz**         |
| `u`            | **增加偏移** | **+1.0 Nm** | **无限制**                |
| `j`            | **减少偏移** | **-1.0 Nm** | **无限制**                |
| **(空格)** | **重置参数** | **-**       | **幅度=0, 频率=1.0 Hz, 偏移=0** |

### 电机选择

#### 非测试模式

| **按键** | **功能**             |
| -------------- | -------------------------- |
| `l`          | **选择左侧电机组**   |
| `r`          | **选择右侧电机组**   |
| `1`-`6`    | **直接选择电机 1-6** |

#### 测试模式

| **按键** | **功能**                  | **说明**                            |
| -------------- | ------------------------------- | ----------------------------------------- |
| `l`          | **选择左侧电机组**        | **切换到左侧电机**                  |
| `m`          | **进入/退出电机选择模式** | **切换后可用** `1`-`6`选择电机  |
| `4`-`9`    | **选择电机 1-6**          | **直接映射：4→1, 5→2, ..., 9→6** |

**电机选择模式使用流程**：

1. **按 **`m` 进入电机选择模式
2. **按 **`1`-`6` 选择对应电机
3. **选择后自动退出电机选择模式**

### 波形参数

**所有波形都支持以下参数：**

* **幅度（Amplitude）**：波形的峰值大小，单位：Nm
* **频率（Frequency）**：波形周期数，单位：Hz（范围：0.1 - 10.0 Hz）
* **偏移量（Offset）**：波形的直流偏移，单位：Nm

**最终输出**：`波形值 + 偏移量`

---

**## **参数说明

### 默认参数

* **波形类型**：方波
* **波形幅度**：1.0 Nm
* **波形频率**：1.0 Hz
* **波形偏移**：0.0 Nm
* **控制周期**：0.002 秒（500 Hz）
* **电机选择**：左侧电机 1

### 电机索引计算

**程序使用以下公式计算实际电机索引：**

```
 motor_idx = select_motor_id + left_right_mode × 6
```

**其中：**

* `select_motor_id`：0-5（对应电机 1-6）
* `left_right_mode`：0（左侧）或 1（右侧）

**例如：**

* **左侧电机 3：**`motor_idx = 2 + 0 × 6 = 2`
* **右侧电机 2：**`motor_idx = 1 + 1 × 6 = 7`

## 数据记录

### 自动记录

**程序启动时会自动启动 rosbag 记录器，记录以下数据：**

* **控制命令**：`/joint_cmd`（包含所有波形控制命令）
* **传感器数据**：`/sensors_data_raw`（电机反馈数据）
* **其他话题**：根据 `log_topics.yaml` 配置记录 70+ 个话题

### Bag 文件位置

* **默认位置**：`~/.ros/`
* **文件命名**：`YYYY-MM-DD-HH-MM-SS_N.bag`
* **文件管理**：
  * **每个文件最大 500 MB**
  * **达到大小限制时自动分割**
  * **最多保留 8 个文件**

### 查看记录的数据

```
 # 查看 bag 文件信息
 rosbag info ~/.ros/2025-11-20-11-49-06_0.bag
 
 # 查看 /joint_cmd 话题内容
 rostopic echo -b ~/.ros/2025-11-20-11-49-06_0.bag /joint_cmd | head -50
 
 # 提取特定话题到新文件
 rosbag filter ~/.ros/2025-11-20-11-49-06_0.bag filtered.bag \
   "topic == '/joint_cmd' or topic == '/sensors_data_raw'"
```

## 使用示例

### 示例 1：测试左侧第3个电机的正弦波

1. **启动程序：**
   ```
    roslaunch hardware_node hardware_testRoban.launch
   ```
2. **等待机器人移动到校准姿态**
3. **选择左侧电机组：按 **`l`
4. **选择电机 3：按 **`3`（非测试模式下）
5. **进入测试模式：按 **`t`
6. **切换到正弦波：按 **`3`（测试模式下）
7. **调整参数（可选）：**
   * **按 **`w` 增加幅度到 2.0 Nm
   * **按 **`e` 增加频率到 2.0 Hz
8. **观察电机运动**
9. **退出测试：按 **`q`

### 示例 2：测试右侧第1个电机的方波（使用电机选择模式）

1. **启动程序并等待校准姿态**
2. **选择右侧电机组：按 **`l` 然后按 `r`（如果当前是左侧）
3. **进入测试模式：按 **`t`
4. **进入电机选择模式：按 **`m`
5. **选择电机 1：按 **`1`
6. **切换到方波：按 **`1`（此时已退出电机选择模式）
7. **开始测试，观察电机运动**

### 示例 3：实时调整波形参数

1. **进入测试模式并选择电机后**
2. **逐步增加幅度：**
   * **按 **`w` 多次，观察电机响应
3. **调整频率：**
   * **按 **`e` 增加频率，观察运动速度变化
   * **按 **`d` 减少频率，观察运动变慢
4. **添加偏移：**
   * **按 **`u` 添加偏移，观察波形整体上移
5. **重置参数：**
   * **按空格键重置所有参数**
