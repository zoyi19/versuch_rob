# 底盘直接控制说明

## 功能介绍

此功能用于直接控制轮臂机器人底盘运动，**不依赖上层控制器**。

适用场景：
- 腰部或手部等硬件损坏，无法启动 kuavo-ros-control 程序时
- 机器人到货后首次移位
- 维修时需要移动底盘

## 控制话题

直接发布到底盘底层接口：`/move_base/base_cmd_vel` (geometry_msgs/Twist)

## 使用方法

### 1. 编译

```bash
cd ~/kuavo_ws
catkin build humanoid_wheel_interface_ros
source devel/setup.bash
```

### 2. 启动控制节点

#### 方式一：使用 bt2 手柄控制

```bash
roslaunch humanoid_wheel_interface_ros chassis_direct_control.launch joystick_type:=bt2
```

#### 方式二：使用 bt2pro 手柄控制

```bash
roslaunch humanoid_wheel_interface_ros chassis_direct_control.launch joystick_type:=bt2pro
```

**手柄操作：**
- **左摇杆**: 控制底盘前后左右移动
- **右摇杆(水平)**: 控制底盘旋转
- **BACK按钮**: 急停

#### 方式三：使用键盘控制

```bash
roslaunch humanoid_wheel_interface_ros chassis_direct_control.launch joystick_type:=keyboard
```

或者直接运行脚本：

```bash
rosrun humanoid_wheel_interface_ros chassis_keyboard_control.py
```

**键盘操作：**
- **w/s**: 前进/后退
- **a/d**: 左移/右移
- **j/k**: 左转/右转
- **空格**: 急停
- **+/-**: 增加/减少速度
- **h**: 显示帮助
- **Esc**: 退出

### 3. 参数调整

可以通过 launch 参数调整速度限制：

```bash
roslaunch humanoid_wheel_interface_ros chassis_direct_control.launch \
    joystick_type:=bt2 \
    linear_scale_x:=0.3 \
    linear_scale_y:=0.3 \
    angular_scale_z:=0.3
```

### 4. 仿真模拟

```bash
export ROBOT_VERSION=60 #或 export ROBOT_VERSION=61
roslaunch humanoid_controllers load_kuavo_mujoco_sim_wheel.launch   joystick_type:=bt2 pure_sim:=true
```
然后参考 `“2. 启动控制节点”` 启动遥控器



### 5. 手柄类型说明

支持两种北通手柄型号：
- **bt2**: 北通2代手柄
- **bt2pro**: 北通2代Pro手柄
