## 说明
- 轮臂系列产品采用 `Kuavo机器人上肢` + `轮式底盘` + `升降模组` 三合一模式组成，同时兼具人形上肢的灵活性以及轮式底盘的快速移动特点

   ![轮臂机器人](./images/轮臂机器人.jpg)

## 产品参数

| 参数名称           | 参数值                          |
|--------------------|---------------------------------|
| 整机尺寸           | 2200\*710\*710mm                 |
| 整机自重           | 245kg                          |
| 高度升降           | 1100mm垂直调节                  |
| 头部自由度         | 2个                            |
| 手臂自由度         | 7个                            |
| 视觉感知传感器     | 2个(底盘1/头1)                 |
| 末端执行器负载     |单臂手最大负载5kg                   |
| 驱动方式           | 四轮万向驱动                   |
| 导航方式           |slam 定位导航        |
| 导航最大速度       | 1.5m/s                         |
| 最小转弯半径       | 0mm(原地转弯)                  |
| 最小通过宽度       | 900mm                          |
| 续航时间           | ≥6h                           |
| 充电时间           | 2h                           |
| 充电方式           | 手动/自动                      |
| 爬坡角度           | 8°                            |
| 全局定位精度       | ±10mm/±3°                     |
| 二次定位精度       | ±5mm/±1°                       |
| 定位激光数量       | 1（2D激光雷达）    |
| 软件支持           | PC端、iPad端软件支持           |
